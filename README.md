# Inspired  React + Redux Toolkit practice project

Учебный проект на React, выполненный в рамках практики современного frontend-стека.  
Проект создан на основе обучающих материалов с целью закрепления навыков работы с React и Redux Toolkit.

## Технологии

- React
- Redux Toolkit
- React Router
- JavaScript (ES6+)
- SCSS
- Vite

## Описание проекта

Проект направлен на практическое изучение и закрепление:

- компонентного подхода React
- управления глобальным состоянием через Redux Toolkit
- работы с асинхронными запросами (createAsyncThunk)
- организации структуры React-приложения
- маршрутизации (React Router)


## Реализованный функционал

- SPA-приложение на React
- Настроенная маршрутизация
- Управление состоянием через Redux Toolkit
- Работа с глобальным store
- Асинхронная загрузка данных через Redux Toolkit thunks
- Разделение на компоненты и страницы
- Базовая логика приложения

## Архитектура

- Компонентный подход React
- Централизованное состояние через Redux Toolkit
- Разделение UI и бизнес-логики
- Структура проекта, приближенная к реальным приложениям


## 📌 Примечание

Проект выполнен в учебных целях на основе обучающих материалов и использован для практики React и Redux Toolkit.
